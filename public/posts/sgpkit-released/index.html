swift-sgp4 1.0.0 is here!

July 3, 2022 One-minute read
Libs

Today I released the first open source version of swift-sgp4, a very simple Swift Package that calculates the position of a satellite given a two-line elements set (TLE).

The library wouldn’t have been possible without the fundamental help of sgp4, a satellite prediction C++ library developed by Daniel Warner.

As you may imagine, swift-sgp4 is the core of [ISSSeeker]({{ site.url }}/apps/issseeker).
